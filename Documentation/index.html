Overview

MoodStrap is built on Twitter bootstrap theme framework and jQuery javascript framework using the nicest parts of HTML5 and CSS3.

The template is fully responsive and built with latest Twitter Bootstrap 3.0.2 framework. It contains many useful jQuery plugins that can be used according to your needs on your application. The template is lightweight and can run flawlessly even on slow machines despite the visual effects. MoodStrap is designed to work on many devices not limited to Desktop resolutions and major browsers but also on Laptops, mobile devices like phones, tablets and many other smart devices.

Font

MoodStrap is using Open Sans font generally which is provided by Google. Markup code for this font is below:

@import "http://fonts.googleapis.com/css?family=Open+Sans:400,600,700&subset=latin,latin-ext";
